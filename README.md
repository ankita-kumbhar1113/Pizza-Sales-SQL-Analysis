# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview
This project analyzes a pizza store's sales data using SQL. It demonstrates how SQL can be used to answer business questions, generate sales insights, and support data-driven decision making.

---

## 🎯 Project Purpose

The objective of this project is to:

- Analyze pizza sales performance using SQL
- Practice writing real-world business queries
- Generate meaningful insights from sales data
- Strengthen SQL skills for Data Analyst and MIS Executive roles

---

## 🛠 Tools Used

- MySQL
- MySQL Workbench

---

## 📂 Database Tables

The project consists of four tables:

- Customers
- Orders
- Pizza Menu
- Order Details

---

## 📊 SQL Concepts Explored

- SELECT Statement
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- INNER JOIN
- Subqueries
- CASE Statement
- HAVING Clause
- Window Functions (RANK, Running Total)
- Date & Time Functions
- LIMIT

## 📷 Sample SQL Queries

### Query 2: Total Revenue Generated

**Business Question:**
What is the total revenue generated from all pizza sales?

**Concepts Demonstrated:**
- Aggregate Function (`SUM`)
- `INNER JOIN`
- Arithmetic Operations
- Business KPI Calculation

> **Result:** Total Revenue = **₹4,650**

### Query 7: Most Ordered Pizza (By Quantity)

**Business Question:**
Which pizza was ordered the most based on quantity sold?

**Concepts Demonstrated:**
- `GROUP BY`
- Aggregate Function (`SUM`)
- `ORDER BY DESC`
- `LIMIT`
- Sales Performance Analysis

> **Result:** **Margherita** was ordered **5** times.

![Query 7](https://github.com/ankita-kumbhar1113/Pizza-Sales-SQL-Analysis/blob/main/query%207%20-%20most%20ordered%20pizza%20by%20quantity.png)

### Query 11: Rank Pizzas by Revenue

**Business Question:**
How do pizzas rank based on the revenue they generated?

**Concepts Demonstrated:**
- Window Function (`RANK()`)
- Aggregate Function (`SUM`)
- `GROUP BY`
- `ORDER BY`
- Revenue Analysis

> **Result:** Margherita generated the highest revenue (**₹950**) and ranked **1st**.

![Query 11](https://github.com/ankita-kumbhar1113/Pizza-Sales-SQL-Analysis/blob/main/query%2011-rank%20pizzas%20by%20revenue.png)

### Query 13: Highest Spending Customer

Which customer spent the highest amount on pizza orders?

**Concepts Demonstrated:**

- `GROUP BY`
- `ORDER BY DESC`
